# Encapsulation(캡슐화)

- 송신 데이터에 필요한 헤더를 붙여 다음 계층으로 보내는 작업
- 헤더에는 각 계층에서 수행한 정보를 포함

방향

- 상위 계층 → 하위 계층

<br>

# Decapsulation(역캡슐화)

- 캡슐화의 반대
- 수신 측의 하위 계층에서 넘겨 받은 정보에 포함된 헤더를 해석? 제거?

방향

- 하위 계층 → 상위 계층
